# Zord-Writer

## Introduction

The Zord-Writer project is a frontend application developed to assist in writing
books using ChatGPT. This project uses the Next.js framework and is built on
React 18.2.0.

## Requirements

To start working with this project, make sure you have installed:

- Node.js (recommended version: v14.0.0 or higher)
- NPM (recommended version: v6.0.0 or higher)

## Installation

1. Clone the repository: https://github.com/edneyreis999/zord-project

2. Enter the project directory: `cd zord-writer`

3. Install the dependencies: `npm install`

## Usage

### Development

To start the development server:

`npm run dev`

This will start the server on port 3000. Access the application in the browser
at `http://localhost:3000`.

### Build

To build the project for production:

`npm run build`

### Execution

To run the built project:

`npm run start`

### Tests

To run the project tests:

`npm run test`

### Storybook

To start the Storybook:

`npm run storybook`

This will start the Storybook on port 6006. Access the Storybook in the browser
at `http://localhost:6006`.

### Lint

To run the project lint:

`npm run lint`

### Type checking

To run type checking with TypeScript:

`npm run type-check`

## Dependencies

Here are some of the main project dependencies:

- `@styled-icons/material-outlined`: Styled icons for the project.
- `next`: Next.js framework for the project.
- `react`: React library.
- `react-dom`: React library for DOM manipulation.
- `react-markdown`: Library for rendering Markdown in React.
- `storybook-addon-next`: Storybook addon for Next.js.
- `styled-components`: Library for styling components.
